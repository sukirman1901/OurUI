# Docs layout

| Path | Role |
|------|------|
| [`user/`](../user/) | End-user guides (Stable APIs) |
| [`decisions/`](../decisions/) | ADRs (accepted local decisions) |
| [`rfcs/`](../rfcs/) | RFCs (Draft → Accepted → Implemented) |
| [`architecture/`](./) | Cross-cutting architecture notes (non-normative sketches) |
| [`../roadmap.md`](../roadmap.md) | Product milestones / releases |

RFCs are **not** roadmap rows. A release may ship without an RFC; an Accepted RFC may span multiple releases.
